# Website UI kit — Code the Future marketing home

A single-page marketing recreation built on the design system primitives.

**Sections:** glass pill `Nav` → `Hero` (eyebrow, big display headline, dual CTAs, trust
row, dark code card) → `Pillars` (3 feature cards) → `ModelStrip` (the dark mental-model
band) → `CampBand` (Spark-gradient CTA) → `Footer`.

**Files:** `index.html` (loads React, Babel, Lucide, the DS bundle, then the JSX),
`site.css` (layout — not tokens), `icons.jsx` (Lucide helper), `Nav.jsx`, `Hero.jsx`,
`Sections.jsx`, `App.jsx`.

Composes `Button`, `Badge`, `Avatar`, `Card` from `window.CodeTheFutureDesignSystem_909f12`.
Tagged as a starting point ("Website").

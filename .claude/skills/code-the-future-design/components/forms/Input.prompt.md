Text field with label, hint, error state and optional leading icon; brand-blue focus ring, finger-friendly height.

```jsx
<Input label="Camper name" placeholder="First & last" hint="As it appears on the form" />
<Input label="Email" type="email" error="That doesn't look right" />
```

Props: `label`, `hint`, `error`, `iconLeft`, plus all native `<input>` attributes.

01 — The mark

The Forward Branch

Code the Future

The story

A path that rises and forks: one branch sprouts to a node — the path you could take — and the other shoots forward into an arrow. It reads three ways at once: a code branch, growth, and forward motion. It deliberately avoids the clichés — no </> brackets, no robots. The icon stands alone as an app icon and avatar.

Clear space & don'ts

Keep clear space equal to the icon's lowest node on all sides. Don't recolor outside the palette, stretch, add effects, or set the wordmark in another typeface. Future sits in brand blue (teal on dark).

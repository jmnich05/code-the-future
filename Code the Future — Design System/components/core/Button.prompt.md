Pill-shaped action button — the primary control; use `primary` for the main path, `accent` (coral) for delight moments, `secondary`/`ghost` for lower emphasis.

```jsx
<Button variant="primary" size="lg" iconRight={<ArrowIcon />}>
  Start building
</Button>
<Button variant="secondary">Explore the camp</Button>
<Button variant="ghost" size="sm">Learn more</Button>
```

Variants: `primary` (blue 600), `accent` (coral), `secondary` (outlined), `ghost`. Sizes: `sm` / `md` / `lg`. Props: `iconLeft`, `iconRight`, `fullWidth`, `disabled`.

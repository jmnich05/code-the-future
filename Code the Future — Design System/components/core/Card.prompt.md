Rounded content container with the brand's soft cool shadow; `dark` uses the Night gradient for futuristic sections.

```jsx
<Card variant="elevated" padding="lg" interactive>
  <h3>Module 02 — The agent loop</h3>
  <p>From a normal loop to an AI agent loop.</p>
</Card>
```

Variants: `default` / `elevated` / `dark` / `outline`. Props: `padding` (sm/md/lg), `interactive` (hover lift).

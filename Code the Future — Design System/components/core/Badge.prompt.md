Mono-cased status/label pill in the brand's operator voice (uppercase JetBrains Mono on a tinted background).

```jsx
<Badge variant="secondary" dot>Live</Badge>
<Badge variant="accent">New</Badge>
<Badge variant="primary">Summer 2026</Badge>
```

Variants: `neutral` / `primary` / `secondary` / `accent` / `success` / `warning` / `danger`. Add `dot` for a leading status dot.

# Code the Future

A platform and in-person summer camp teaching kids AI literacy and modern coding — with
special attention to how AI is reshaping software development.

**Two things, one mission:**

- **Platform** — a web product for learning AI + coding the way the work actually happens.
- **Summer camp (2026)** — hands-on, in-person: what AI is, how to use it well, and how it
  changes coding.

## Teaching philosophy

Applied, operator-grade, AI-native. We teach **Python + TypeScript + SQL** together and
tie every concept to a real loop. The model students internalize:

> Code handles the loop. AI handles the judgment inside the loop. Rules handle the
> guardrails. Humans handle exceptions.
